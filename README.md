# Customer Retail Analysis Project

A data analysis project focused on customer retail behavior and insights.

## Project Structure

- `customer_retail.ipynb` - Main analysis notebook
- `data/` - Data files directory
- `requirements.txt` - Python dependencies

## Setup

1. Install dependencies:
   ```
   pip install -r requirements.txt
   ```

2. Run the Jupyter notebook:
   ```
   jupyter notebook customer_retail.ipynb
   ```

## Dependencies

- pandas
- jupyter
